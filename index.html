Description du Jeu Cookie Clicker

Titre : Cookie Clicker

Genre : Jeu de clic (Clicker Game), Incremental Game

Plateforme : Navigateur web (HTML5), disponible également sur mobile (iOS, Android)

Développeur : Julien "Orteil" Thiennot

Date de sortie initiale : Août 2013

Concept de base :

Cookie Clicker est un jeu de type incremental où le joueur clique sur un gros cookie affiché à l'écran pour produire des cookies virtuels. Le but du jeu est de produire autant de cookies que possible en cliquant sur le cookie géant et en achetant des améliorations qui augmentent la production automatique de cookies.

Mécanismes du jeu :

  • Cookie géant : Le joueur commence le jeu avec un seul cookie géant qu'il peut cliquer pour produire des cookies.
  • Chaque clic : Chaque clic sur le cookie géant augmente le nombre de cookies accumulés.
  • Améliorations : Les joueurs peuvent acheter des améliorations qui augmentent la production de cookies automatiquement, même lorsque le joueur n'est pas actif.
  • Objectif : L'objectif est de maximiser la production de cookies et d'explorer les différents niveaux d'améliorations disponibles.
